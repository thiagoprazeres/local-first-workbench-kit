# Changelog

## 0.0.1

Initial release.

- Prompt protocol executor with sessions, steps, step results, and evaluations.
- Sequence runner for ordered protocols.
- Prompt sources: memory, file, composite.
- Step contracts with pluggable validators.
- JSON session serialization.
- Typed errors.
- Provider-agnostic adapter interface.
- ESM, TypeScript declarations, zero runtime dependencies.
